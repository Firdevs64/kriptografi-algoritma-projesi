# Kriptografik Algoritma Geliştirme ve Analizi

Bu depoda (repository), temel kriptografik prensipleri kullanarak geliştirilen bir blok şifreleme algoritmasının:
- tasarımı (Aşama 1),
- Python ile kodlanması ve test edilmesi (Aşama 2),
- kriptanalizi ve değerlendirilmesi (Aşama 3)
yer almaktadır.

## Özellikler
- Algoritma Tipi: Blok Şifre
- Blok Boyutu: 16 byte (128 bit)
- Anahtar Boyutu: 8 byte (64 bit)
- Temel İşlemler: XOR, İkame (S-Box), Permütasyon

## Proje Yapısı
- `src/sifreleme_algoritmasi.py` : Şifreleme / deşifreleme kodu
- `tests/test_sifreleme.py` : Doğrulama ve anahtar hassasiyeti testleri (pytest)
- `rapor/asama1_tasarim_raporu.md` : Tasarım raporu ve şema
- `rapor/asama3_kriptanaliz_raporu.md` : Kriptanaliz ve sonuçlar

## Çalıştırma
Proje ana dizinindeyken:
```bash
python src/sifreleme_algoritmasi.py
